Quick Sort

This sorting algorithm follows the principle of divide and conquer. It picks an element in the list (traditionally the first or last) and considers it a partition that is used to place items of the list that are of smaller value into a new list and items of larger value into a new list . This partition process is repeated until the newly created list of partitioned items are an individual item and the list is reconstructed from the individual items in a sorted order.

Process

1. Find your pivot position using the partition

2. Place items in the list that are smaller in the partition in a new list and items that are larger in a new list.

3. Repeat steps 1-2 until the list is split into individual items based on newly created partitions from the created lists.

4. Reassemble the list in the ordered fashion.

Complexities

Time Complexity - O(Log(N))
