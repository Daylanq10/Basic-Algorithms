Bubble Sort

This sorting algorithm compares two adjacent items in list and checks which one is smaller, if the one on the right is smaller, then the items swap places. It then moves down the list and compares each adjacent pairs and continuously passes until all items are sorted.

Process

1. Compare the first two items in the list, if they are not ordered, then swap their positions in the list.

2. Move one index down the list and compare the next two items, again comparing and swapping if they are not in order.

3. Continue down the list until all adjacent items have been compared.

4. Repeat steps 1-3 until every compared items in the list are in the correct order

Complexities

Time Complexity - O(N^2)
