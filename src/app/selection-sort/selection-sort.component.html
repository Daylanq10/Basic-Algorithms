Selection Sort

This sorting algorithm used an original unsorted list which it iterates through and creates a sub list of the sorted elements

Process

1. Iterate through the original list and place the minimum item in the new sorted list.

2. Remove the minimum item from the original list.

3. Repeat steps 1 and 2 until the original list is empty and the sorted list contains all sorted items.

Complexities

Time Complexity - O(N^2)
