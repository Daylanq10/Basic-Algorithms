Insertion Sort

This sorting algorithm iterates through the list and compare the current item to previous items in the list until the list is ordered.

1. Iterate from the start of the list.

2. As you iterate through items, compare the current item to its previous item, if the prevous item is larger, then swap them.

3. The current item will continue to compare previous items and swap until the previous item is no longer larger than it.

4. The list then iterates to a new current item at the index after the original position of the previous current item.

5. The process ends once the entire list has been iterated through and will leave a sorted list.

Complexities

Time Complexity - O(N^2)
