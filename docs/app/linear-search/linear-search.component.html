Linear Search

This search algorithm is a simple approach to finding an item in an array or list that iterates each individual item until a result is found or not found

Process

1. Start with the left most item in the list.

2. Iterate across the list until the desired item is found.

3. If the item is found, return the index.

4. If the item is not in the list, return -1.

Complexities

Time Complexity - O(N)
